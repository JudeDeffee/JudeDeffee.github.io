ABOUT MY DUNGEON GENERATOR
Volcano island is a sim game with the goal to grow a resort island empire, attracting tourists and managing resources. Players start with a small island and limited funds, gradually expanding their resort by constructing various buildings, amenities, and attractions to entice visitors. The game features a unique island system where players use volcanic eruptions to create new landmasses, adding a strategic layer to island expansion.
MY CONTRIBUTIONS
I picked up this game post initial launch, with the objective to improve and expand the game based on user feedback. My contributions included optimizing performance to ensure smooth gameplay on mobile devices, implementing new features such as additional buildings and attractions, and enhancing the user interface for better player experience. I also worked on bug fixes and balancing gameplay mechanics to improve overall enjoyment. Additionally i worked on monitizing the game where IAPs where largely absent before my involvement.
MY PROUDEST ADDITIONS
One of my proudest additions to Volcano Island was the implementation of a mesh combining system, which significantly improved the game's performance on mobile devices. By reducing the number of draw calls, I was able to enhance frame rates and overall gameplay smoothness.
