ABOUT THE GAME
The game is a premium mobile game targetted at young children. The game is designed to be both educational and entertaining, featuring a variety of mini-games and activities that promote learning through play. The game will feature multiple minigames each tailored to kids, taking inspiration from Marvel HQ. The game has a kid friendly layout and art style, with bright colours and engaging imagry.
MY CONTRIBUTIONS
I programmed a jigsaw puzzle minigame from scratch using Unity and C#. Most notably the jigsaw puzzle is procedurally generated, using a source image combined with a custom compute shader i programmed to create puzzle pieces relavant to their position, with only two images to allow for quick styling and modification. Alongside this I implemented a piece seperation mechanic allowing for pieces to push each other appart, to achieve this in a performant way i learnt about spatial partitioning and implemented a grid based system to only check collisions between nearby pieces.
