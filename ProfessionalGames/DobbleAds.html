ABOUT PLAYABLE DOBBLE AD
This was a project involving the creation of playable ads for the app Dobble Go. The ads were designed to engage users by allowing them to interact with a mini-game that showcased the features of Dobble Go. The playable ad was developed using HTML, CSS, and JavaScript, ensuring compatibility across various devices and platforms. The goal was to provide an immersive experience that would entice users to download and try out the full game.
MY CONTRIBUTIONS
This was my first experience working with HTML JS and CSS. I learnt how to create interactive web content and optimize it for performance within two weeks, programming my own physics and particle system. Playable ads require that all assets are inline, and therfore i learned about encoding images and audio into base64 strings to be used within the code. I also had to learn about the limitations of playable ads, such as file size restrictions and performance constraints, and how to work within these limitations to create an effective ad.
